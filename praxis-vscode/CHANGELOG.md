# Changelog

All notable changes to the Praxis VS Code extension will be documented in this file.

## [0.1.0] - 2026-01-20

### Added
- Initial release
- `praxis.start` command to launch Praxis in terminal
- `praxis.startInSplit` command for split terminal view
- Keyboard shortcuts (`Cmd+Shift+;` and `Cmd+Escape`)
- Configuration settings for executable path, models, and debug mode
- Terminal cleanup on extension deactivation
